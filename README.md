# DreamOS

DreamOS is the canonical Dream.OS runtime, transport, contract, and swarm execution repository.

## Role
This repo owns the executable core for:

- message contracts
- transport envelopes
- execution guards
- relay and claim flows
- runtime task execution
- swarm routing and core agent behavior
- audit and regression gates

## Canonical Boundary
- DreamOS owns runtime/swarm execution.
- AgentTools owns operator tooling.
- DreamVault owns governance, inventory, reporting, and promotion manifests.
