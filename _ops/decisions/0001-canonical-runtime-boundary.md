# 0001 - DreamOS Canonical Runtime Boundary

## Decision
DreamOS is the canonical Dream.OS runtime, transport, relay, schema, and execution-guard repository.

## DreamOS Owns
- message contracts
- execution guards
- relay and transport flows
- runtime semantics
- audit gates

## DreamOS Does Not Own
- operator dashboards
- Discord control-plane tooling
- governance inventory
- marketing automation

## Canonical Owners
| Concern | Repository |
|---|---|
| Runtime/swarm execution | DreamOS |
| Operator/control-plane tooling | AgentTools |
| Governance/inventory/marketing | DreamVault |
